What is an API key?
Tag Engine is powered by Claude — an AI made by Anthropic. To use it, you connect your own Anthropic account via an API key. Think of it as a personal access pass. Your key stays in your browser and is never shared.
Why not use a shared key?
Using your own key means you only pay for what you use, your usage stays private, and the tool stays free to share with no one footing the bill for everyone else.
What does it cost?
Each generation costs roughly $0.006 — less than one cent. You can run it 1,000 times for about $6. No subscription. Pay-as-you-go only.
How to get your key — 5 steps
01Go to console.anthropic.com and create a free account.
02Go to Settings → Billing and add a card. Load $5–$10 to start.
03Go to API Keys in the sidebar → Create Key. Name it anything.
04Copy the key immediately — you only see it once. It starts with sk-ant-
05Paste it in the field above and hit Save. Done.
Never share your key publicly. If it gets exposed, revoke it at console.anthropic.com and generate a new one.
